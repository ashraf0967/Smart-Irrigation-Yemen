# دليل المستخدم لنظام الري الذكي

## 1. إعداد النظام
1. قم بتوصيل جميع المكونات كما في دليل إعداد العتاد
2. شحن البطارية بالكامل قبل التشغيل الأول
3. تحميل البرنامج على متحكم ESP32 باستخدام Arduino IDE

## 2. التشغيل الأولي
1. تأكد من وجود تغطية شبكة LoRa/GSM
2. شغل النظام باستخدام المفتاح الرئيسي
3. انتظر حتى يبدأ المؤشر في الوميض (يدل على اكتمال الإقلاع)

## 3. استخدام لوحة التحكم
### شاشة الرئيسية
- عرض لمستويات رطوبة التربة في الوقت الحقيقي
- قراءات درجة الحرارة والرطوبة الجوية
- حالة البطارية ومستوى الشحن

### شاشة التحكم
- أزرار تشغيل/إيقاف لكل صمام
- ضبط مدة الري اليدوي
- عرض حالة الصمامات الحالية

### شاشة الجدولة
- إضافة جدول ري يومي أو أسبوعي
- تحديد وقت البدء والمدة
- تفعيل/تعطيل الجداول

## 4. الصيانة الدورية
1. تنظيف حساسات التربة كل أسبوعين
2. فحص وصلات الأسلاك شهرياً
3. تنظيف اللوح الشمسي من الأتربة
4. استبدال البطارية كل 2-3 سنوات
